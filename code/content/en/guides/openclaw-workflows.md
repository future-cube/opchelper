This guide focuses on three repeatable loops:
1) Content: topic → draft → publishing checklist → review
2) Leads: lead → outreach → follow-up cadence → post-mortem
3) Delivery & cashflow: brief → acceptance → change requests → payment reminders

Start read-only, write outputs into a dedicated `outputs/` folder, and only move verified results into your real repo.
