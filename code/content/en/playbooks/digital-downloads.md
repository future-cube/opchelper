## Principle: validate before building

Most failures happen because people build first and validate later.

## Minimal loop

1) Pick a high-frequency niche scenario  
2) Create an example output (what buyers will get)  
3) Write a quick-start guide (10–20 minutes)  
4) List it on one platform (don’t spread too early)  
5) Use content to drive traffic  
6) Collect feedback → v1.1
