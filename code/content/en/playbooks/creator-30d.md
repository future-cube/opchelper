This playbook is built around one principle: **run the shortest loop first**.

**Loop:** publish → capture a lead → deliver one concrete result → get paid → review → iterate.

## Week 1: Positioning

- Write a one-liner: “I help [who] in [context] solve [problem] to get [result].”
- Pick one primary monetization model (service / digital download / membership / tool).

## Week 2: Content system (3 recurring series)

- Problem posts (distribution)
- Case posts (trust)
- Template posts (conversion)

Every post should have a next step: download, DM, email list, or a product page.

## Week 3: Offer + delivery

- Define the result you sell (not “hours”).
- Create a simple deliverable checklist and acceptance criteria.

## Week 4: Review + scale

- Weekly review: output, metrics, bottleneck, next focus (1–2 items).
- Expand in order: acquisition → delivery → cash-flow → scaling.
