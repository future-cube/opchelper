# Newsletter 欢迎序列模板（3 封）

> 目标：用 3 封邮件完成“自我介绍 → 建立信任 → 引导行动”。

## 邮件 1（第 0 天）：欢迎与预期

主题：欢迎加入｜你将从这里得到什么

内容：
- 我是谁（1 段）
- 你将收到什么（栏目/频率/价值）
- 一个简单问题：你最想解决什么？

CTA：
- 回复我：你的现状/目标是什么？

## 邮件 2（第 2 天）：高价值方法/案例

主题：一个能立刻用上的方法（附模板/清单）

内容：
- 一个案例/方法（结构化）
- 关键步骤（3–5 步）
- 模板/清单链接

CTA：
- 下载/收藏模板

## 邮件 3（第 5 天）：明确行动（产品/服务）

主题：如果你想更快达成结果：这里是下一步

内容：
- 适合谁/不适合谁
- 你提供的结果是什么
- 如何开始（链接/预约/购买）

CTA：
- 预约/购买/加入
