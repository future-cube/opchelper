# 一人公司每周复盘清单（30 分钟）

> 固定在同一时间做复盘：比如每周五 18:00。把复盘当作“经营例会”。

## 1. 本周产出（事实）

- 我发布/交付了什么？
- 我完成了哪些关键动作（获客/交付/回款）？

## 2. 本周指标（数字）

- 曝光/访问：
- 点击/收藏/私信：
- 线索数量：
- 成交数量：
- 回款金额：

## 3. 本周瓶颈（只写最关键 1–2 个）

- 卡在选题/写作/转化/交付/回款中的哪一环？
- 这个瓶颈的根因是什么？

## 4. 下周重点（只选 1–2 项）

- 下周最重要的一个目标：
- 为此我必须完成的 3 个动作：
  - [ ] 
  - [ ] 
  - [ ] 

## 5. 复盘结论（可复用）

- 本周有效的做法（下次继续）：
- 本周无效的做法（下次停止）：
